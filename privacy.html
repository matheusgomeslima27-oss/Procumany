Política de Privacidade

Este aplicativo utiliza a API do Instagram apenas para responder mensagens e comentários automaticamente quando o usuário interage com o perfil.

Nenhuma informação pessoal é armazenada, vendida ou compartilhada com terceiros.

Os dados utilizados são apenas:

  • ID do usuário
  • Mensagem enviada
  • Comentário feito

Esses dados são usados somente para permitir o funcionamento do bot.

Caso tenha dúvidas, entre em contato pelo Instagram do próprio projeto.
